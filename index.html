this is me
I'm Ania, a professional editor and translator, with love for cooking and travelling, currently on a very intensive track to learn web development. Happy to share some of my latest projects with you. More under development, so make sure to check in again soon! If you'd like to leave any feedback or comments, feel free to get in touch via social media! Also, check out the links section - I cherry-picked top resources I had the good luck to find during the first year of learning. Thanks for stopping by!
Favorite web-dev resources
  • FreeCodeCamp – free online bootcamp for front end beginners
  • CS50 – an absolutely fantastic introduction to computer science in general, I strongly recommend it as a supplement to FCC
  • CodeSchool – beautifully prepared web dev courses, some lessons available for free
  • You Don’t Know JS – a free e-book series with an in-depth take on JavaScript, written in a fun and easy style
  • CodeWars – excellent algorithms practice with thousands of challenges and varying difficulty level
  • Git starter - a super-useful guide for making your first Github pull request, great for absolute beginners
  • CodePen – useful tool for putting online your first web projects
  • JS 30 Day Challenge - tips & tricks on achieving fun or useful effects anf functionalities with vanilla JS.
